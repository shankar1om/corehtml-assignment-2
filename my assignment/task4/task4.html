List of HTML Tags:

  1. !DOCTYPE :
    • DOCTYPE tag is used to inform the browser about the version of HTML used in the document.
  2. anchor tag
    • anchor tag defines a hyperlink that links one page to another page.
      • href attribute:The href attribute is used to define the address of the file to be linked.
      • target attribute:this Specify a location for Link.
  3. Head
    • The element is used as a container for metadata .
      • title:The HTML element is used to define the title of the document.
      • style:The HTML element is used to style the HTML page.
      • meta: The HTML element is used to specify the character set,
  4. HTML Formatting tag
    • HTML Formatting is a process of formatting text for better look and feel.
      • b tag:This is a physical tag, which is used to bold the text written between it.
      • strong tag:This is a logical tag, which tells the browser that the text is important.
      • I tag: This is a physical tag which is used to make text italic.
      • mark:This tag is used to highlight text.
      • u: This tag is used to underline text written between it.
      • Sup tag:It displays the content slightly above the normal line
      • Sub tag:It displays the content slightly below the normal line.
  5. HTML Heading
    • A HTML heading or HTML h tag can be defined as a title or a subtitle which you want to display on the webpage. When you place the text within the heading tags .
      • h1 tag:This is main heading of page.
      • h2:This is first sub-heading.
      • h3:This is Second sub-heading.
  6. HTML Paragraph
    • HTML paragraph or HTML p tag is used to define a paragraph in a webpage.
  7. HTML Table tag
    • HTML table tag is used to display data in tabular form .
      There can be many columns in a row.
      • tr tag:It defines a row in a table.
      • th tag:It defines a header cell in a table.
      • td:It defines a cell in a table.
      • tfooter:It is used to group the footer content in a table.
  8. HTML list
    • HTML Lists are used to specify lists of information. All lists may contain one or more list elements.
      • ol(ordered list tag): It isused as numbered list .
      • ul(unordered list tag):it is used as list without number.
